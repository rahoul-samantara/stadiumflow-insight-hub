# StadiumFlow Insight Hub

Welcome to the StadiumFlow Insight Hub repository!

## Security & Accessibility
This project contains elements to ensure security and accessibility:
- **Dependabot**: Automatically checks for dependency updates.
- **Security Policy**: See `SECURITY.md` for our vulnerability reporting policy.
- **Code of Conduct**: See `CODE_OF_CONDUCT.md` to understand our community standards.
- **Contributing Guidelines**: Check `CONTRIBUTING.md` for information on how to contribute securely.
- **Ignored sensitive files**: Configured `.gitignore` ensures sensitive credentials (like `.env`) are not exposed.

## Getting Started

1. Clone this repository.
2. Install dependencies (e.g. `npm install` or `bun install`).
3. Run the development server (e.g. `npm run dev` or `bun run dev`).
