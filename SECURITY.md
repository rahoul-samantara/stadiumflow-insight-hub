## Reporting a Vulnerability

Please report (suspected) security vulnerabilities to the maintainers privately rather than opening a public issue. You can do this by sending an email to the repository owner or using GitHub Security Advisories.

We will acknowledge receipt of your vulnerability report and send you regular updates about our progress. If a fix is accepted, we will coordinate with you to publish a security advisory.
